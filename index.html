KAIST SPARCS 2023 StartUp Hackathon

"우리의 생각이 한계까지 함께하는 곳, SPARCS 2023 StartUp Hackathon"
SPARCS상 (상금 1,000,000원)

KAIST SPARCS 2023 StartUp Hackathon은 전국 각지의 기획자, 개발자, 디자이너가 만나 전문 개발자, 투자자의 피드백을 통해 실제 서비스까지 만들어 볼 수 있는, 해커톤과 멘토링 & 데모데이가 융합된 행사입니다.

더보기

개요 :

SPARCS 2023 StartUp Hackathon (KAIST 주최 PRIMER, VoyagerX 등 후원)에 참여하여 온라인 회의에서 실시간 질의응답, 회의 요약, 기록이 가능한 개인화 AI 비서 개발하였습니다.

상세 업무 내용 :

- OpenAI text-embedding-ada-02 모델을 사용하여 명령어 embedding 후, 미리 설정된 명령어와의 vector similarity 계산을 통한 AI 비서와 사용자와의 원활한 상호작용을 가능하게 하였습니다.
- Google Speech-to-Text api를 사용하여 실시간 회의 음성을 텍스트로 전환하고 기록하였습니다.
- Bart-large-xsum 요약 모델을 사용하여 실시간 회의 내용 요약을 구현하였습니다.
- ChatGPT api (davinci-003 model)를 사용하여 회의록을 기반으로 AI 비서와 자유로운 질의 응답 이 가능하도록 구현하였습니다.
- Google Calander와의 연동을 통해 AI 비서가 자동으로 회의에서 정해진 스케줄을 관리 할 수 있게 하였습니다.
- 전체적인 Pipeline 구상 및 api 통합을 진행하여 하나의 서비스로 제작하였습니다.
